### 🚀 Apa yang sedang saya kerjakan:
* 🏥 **Manajemen Linen RS:** Mengembangkan sistem inventaris berbasis **RFID** menggunakan Laravel.
* 🧹 **KlinPluss:** Membangun platform pemesanan layanan kebersihan yang user-friendly.
* 📱 **Exploration:** Sedang mendalami **React Native** untuk menciptakan pengalaman mobile yang mulus.

### 🛠️ Keahlian & Ketertarikan:
* **Backend & Frontend:** Sangat antusias dengan ekosistem **Laravel** dan **Filament**.
* **UI/UX Design:** Gemar mengeksplorasi **Figma** untuk memastikan estetika bertemu dengan fungsionalitas.
* **Collaboration:** Terbuka untuk diskusi mengenai arsitektur web atau kolaborasi proyek open-source.
